# Igrzyska_Olimpijskie
Analiza zbioru danych pochodzących z Kaggle.

Podgląd projektu:
https://akotwicka.github.io/Igrzyska_Olimpijskie/

Analiza eksploracyjna zbioru danych dotyczących Nowożytnych Igrzysk Olimpijskich począwszy od igrzysk w Atenach w 1896 roku, do tych w Rio w 2016 roku.

Analizę przeprowadzono w językach programowania Python i SQL z wykorzystaniem technologii Spark i Hive.
